####1 导言
```
1.1 入门
过程：编写程序文本-》编译，加载，运行，输出。
C语言程序由函数和变量组成：
	函数：指定要执行的计算操作
	变量：存储计算过程中使用的值
main函数为主函数，是必须的。
使用""阔起来的是字符串或字符串常量。
在C语言中\n表示输出换行，它是一个字符（转义字符）
```

```
1.2  变量和算术表达式
//为C语言中的注释，注释介绍了该程序的作用，注释的内容将被编译器忽略。
在C语言中，所有的变量都必须先声明后使用。通常在函数的起始处，任何可执行语句之前。
声明用于说明变量的属性(由类型名和变量表组成)。

int 	通常为16位 表示整数
float   通常为32位，至少有6位有效数字，表示浮点数（可以带有小数部分的数）
char    字符（一个字符）
short   短整型
long    长整型
double  双精度浮点型

C语言中各条语句均以分号结束。
while循环（先判断，如果条件为真则执行语句，否则循环结束）
正确缩进&保留适当空格对程序的易读性非常重要，建议每行一条语句，在运算符两边加上空格。
C语言中，整数除法将执行舍位，结果中的任何小数部分都会被丢弃。
printf函数，通用的输出格式化函数。
注意：printf并不是C语言本身的一部分，C语言本身并没有定义输入和输出功能。它仅仅是标准库函数中一个有用的函数而已。
%3d %6d可以指定打印的宽度，实现输出对齐功能。
%3.0f 表示打印的浮点数至少占3个字符宽度，且不带小数点和小数部分
%6.1f 表示打印的浮点数至少占6个字符宽度，且小数点后面有一位数字。
%.2f  表示按照浮点数打印，小数点后面有两位小数。

%o 八进制打印
%x 十六进制打印
%c 打印字符
%s 打印字符串
%% 打印%符号

运算中所有数据均为整型则执行整型运算，如果浮点型和整型间杂，则会将整型转换为浮点型计算。
```

```
1.3 for语句
在允许使用某种类型变量值的任何场合，都可以使用该类型的更复杂的表达式。
for循环是对while循环的推广，更简单。
for循环表达式：for(int i=0;i<100;i++){}
for循环由四部分组成 for(初始化部分；条件部分；增加步长)执行语句
for循环比较适合初始化和增加步长都是单条语句并且逻辑相关的情形，因为它把循环控制语句放在集中放在一起，并且比while语句更加的紧凑。
```

```
1.4 符号常量
define可以将符号常量定义为一个特殊的字符串，替换文本可以是任何的字符序列。
定义格式：#define 名字 替换文本
示例：#define STEP    200   //增加步长  其中STEP称为符号常量（宏），通常用大写字母拼写

```

```
1.5 字符输入|输出
无论文本从何处输入，输出到何处，其输入|输出都是按照字符流的方式处理的。
文本流是由多行字符构成的字符序列，而每行字符则由0个或多个字符组成，末尾是换行符。
标准库提供的一次读|写一个字符的函数。getchar和putchar.
getchar函数从文本流中读入下一个输入字符，并将其作为结果返回。
```
